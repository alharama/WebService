How to use the web service
  • Right now the web service isn't working correctly on the droplet but works well on the local host if you would like to pull the repository files
  • Operations include the following
  • POST: '/:song': Takes a song with the artist and genre in the json body and adds it to the database
  • !!!!SONGS ARE CASE SENSITIVE AND MUST USE UNDERSCORES FOR SPACES!!!!
  • An example for All of the Lights by Kanye West:
  • curl --request POST --header 'Content-Type: application/json' --data '{"Kanye West":"Rap"}' http://localhost:8443/:All_Of_The_Lights
  • Other functions include:
  • GET '/songs' which returns a list of all song info in the database
  • GET '/:song' returns the song information for the specified song
  • GET '/songs/:artist' returns all songs by the specified artist
  • GET '/songs/artists' returns a list of all artists in the database
  • GET 'songs/music/:genre' returns a list of all the songs of the specified genre in the database
  • PATCH '/:song/favorite' adds a favorite to the specified song
  • DELETE '/:song' deletes the specified song from the database
  • DELETE '/songs/:artist' deletes all songs by the specified artist
