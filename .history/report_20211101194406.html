My Web Service Report

Endpoint Explanation:

  • Operations include the following:
  • !!!!SONGS AND ARTISTS ARE CASE SENSITIVE AND MUST USE UNDERSCORES FOR SPACES!!!!
  • POST: '/:song': Takes a song with the artist and genre in the json body and adds it to the database
  • An example for All of the Lights by Kanye West:
  • curl --request POST --header 'Content-Type: application/json' --data '{"Kanye West":"Rap"}' https://project2.malharazi.me:8443/:All_Of_The_Lights
  • Other functions include:
  • GET '/songs' which returns a list of all song info in the database
  • curl https://project2.malharazi.me:8443/songs
  • GET '/:song' returns the song information for the specified song
  • curl https://project2.malharazi.me:8443/:All_Of_The_Lights
  • GET '/songs/:artist' returns all songs by the specified artist
  • curl https://project2.malharazi.me:8443/songs/:Kanye_West
  • GET '/songs/artists' returns a list of all artists in the database
  • curl https://project2.malharazi.me:8443/songs/artists
  • GET 'songs/music/:genre' returns a list of all the songs of the specified genre in the database
  • curl https://project2.malharazi.me:8443/songs/music/:Rap
  • PATCH '/:song/favorite' adds a favorite to the specified song
  • curl https://project2.malharazi.me:8443/:All_Of_The_Lights/favorite
  • DELETE '/:song' deletes the specified song from the database
  • curl --request DELETE --header 'Content-Type: application/json' https://project2.malharazi.me:8443/:All_Of_The_Lights
  • DELETE '/songs/:artist' deletes all songs by the specified artist
  • curl --request DELETE --header 'Content-Type: application/json' https://project2.malharazi.me:8443/songs/:Kanye_West
